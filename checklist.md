# 里程碑 M2｜验收清单 (Checklist)

## 1. 场地与生成
- [ ] 游戏场地准确显示为 10x10 网格。
- [ ] 每轮准确生成 3 个备选积木。
- [ ] 必须放完当前 3 个备选积木后，才会刷新下一轮。

## 2. 积木与放置
- [ ] 新增 1x4 积木，支持横向与纵向放置。
- [ ] 新增 1x5 积木，支持横向与纵向放置。
- [ ] 长条积木在边界位置可正确判定合法/非法放置。
- [ ] 所有积木均不可与已占据格重叠，且不可越界。

## 3. 消除与计分
- [ ] 10x10 棋盘下，行消除与列消除逻辑正确。
- [ ] 支持同时消除多行、多列及行列交叉。
- [ ] 放置得分准确为积木占用格数 `n`。
- [ ] 消除奖励准确为 `(消除行数 + 消除列数) × 10`。

## 4. 结束与重开
- [ ] 当备选区所有积木都无合法放置点时，准确触发 Game Over。
- [ ] Game Over 弹层展示正常，重开按钮可用。
- [ ] 点击重开后棋盘、分数、备选区状态完整重置。

## 5. 视觉体验（M2 重点）
- [ ] 拖拽过程平滑，无明显跳帧、闪烁、错位。
- [ ] 合法/非法落点预览清晰，状态切换自然。
- [ ] 放置、消除、补货动画节奏统一且观感流畅。
- [ ] 10x10 布局下网格、色彩、层级保持清晰美观。
