# 里程碑 M3｜验收清单 (Checklist)

## 1. 模式与范围
- [x] 游戏为单机 Endless 模式，主流程无关卡分支。
- [x] 不包含社交分享、排行榜与商业化入口。
- [x] 不包含技能、道具、局内变体与成长养成模块。

## 2. 场地与备选区
- [x] 游戏场地准确显示为 10x10 网格。
- [x] 每轮准确生成 3 个备选积木。
- [x] 必须放完当前 3 个备选积木后，才会刷新下一轮。
- [x] 备选区同时出现多个 1x5 长条时无重叠、无遮挡。

## 3. 积木与放置
- [x] 1x4 积木支持横向与纵向放置。
- [x] 1x5 积木支持横向与纵向放置。
- [x] 基础积木（1x1、1x2/1x3、2x2/3x3、L 形）可正常生成与放置。
- [x] 所有积木均不可与已占据格重叠，且不可越界。
- [x] 长条积木在边界位置可正确判定合法/非法放置。

## 4. 消除与计分
- [x] 行消除与列消除逻辑正确。
- [x] 支持多行、多列及行列交叉的同时消除。
- [x] 放置得分准确为积木占用格数 `n`。
- [x] 消除奖励准确为 `(消除行数 + 消除列数) × 10`。
- [x] 连击触发不会增加额外积分，只影响反馈表现。
- [x] 长局中分数累计无异常跳变、重复结算或漏算。

## 5. 结束与重开
- [x] 当备选区所有积木都无合法放置点时，准确触发 Game Over。
- [x] Game Over 弹层展示正常，重开按钮可用。
- [x] 点击重开后棋盘、分数、备选区、连击状态完整重置。
- [x] Game Over 判定在“成功放置后”和“备选补货后”都准确执行。

## 6. 拖拽与预览交互
- [x] 拖拽过程平滑，无明显跳帧、闪烁、错位。
- [x] 合法/非法落点预览清晰，状态切换自然。
- [x] 鼠标拖拽时指针与积木贴合，抓取成功率稳定。
- [x] 移动端触控拖拽时积木上移，减少手指遮挡。
- [x] 触控上移偏移量在不同屏幕尺寸下表现一致。
- [x] 触控偏移不影响落点预览、合法性判定与释放放置结果。

## 7. 视觉、音效与触觉反馈
- [x] 放置、消除、补货动画节奏统一，整体观感流畅。
- [x] 连击视觉反馈具备层级变化，断连后重置正确。
- [x] 连击提示不遮挡核心落点区域，不影响操作判断。
- [x] 落子音、消除音、连击音阶触发正确，连续触发不失真。
- [x] 音效开关、音量档位可用且刷新后保持。
- [x] 支持振动设备可触发差异化反馈，不支持设备平滑降级。
- [x] 振动开关可用且关闭后不再触发触觉反馈。

## 8. 布局与跨端一致性
- [x] 棋盘宽度与顶部信息区、底部备选区视觉比例协调。
- [x] 10x10 布局下网格、配色、层级保持清晰可读。
- [x] 移动端与桌面端关键交互体验一致，无阻断性问题。
