# 里程碑 M2｜验收清单 (Checklist)

## 1. 场地与生成
- [x] 游戏场地准确显示为 10x10 网格。
- [x] 每轮准确生成 3 个备选积木。
- [x] 必须放完当前 3 个备选积木后，才会刷新下一轮。

## 2. 积木与放置
- [x] 新增 1x4 积木，支持横向与纵向放置。
- [x] 新增 1x5 积木，支持横向与纵向放置。
- [x] 长条积木在边界位置可正确判定合法/非法放置。
- [x] 所有积木均不可与已占据格重叠，且不可越界。

## 3. 消除与计分
- [x] 10x10 棋盘下，行消除与列消除逻辑正确。
- [x] 支持同时消除多行、多列及行列交叉。
- [x] 放置得分准确为积木占用格数 `n`。
- [x] 消除奖励准确为 `(消除行数 + 消除列数) × 10`。

## 4. 结束与重开
- [x] 当备选区所有积木都无合法放置点时，准确触发 Game Over。
- [x] Game Over 弹层展示正常，重开按钮可用。
- [x] 点击重开后棋盘、分数、备选区状态完整重置。

## 5. 视觉体验（M2 重点）
- [x] 拖拽过程平滑，无明显跳帧、闪烁、错位。
- [x] 拖拽时鼠标/手指与积木“黏住”，无偏移现象。
- [x] 合法/非法落点预览清晰，状态切换自然。
- [x] 放置、消除、补货动画节奏统一且观感流畅。
- [x] 10x10 布局下网格、色彩、层级保持清晰美观。
