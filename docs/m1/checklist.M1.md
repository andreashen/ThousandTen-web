# 里程碑 M1｜验收清单 (Checklist)

## 1. 场地与生成
- [x] 游戏场地准确显示为 9x9 的网格。
- [x] 每次准确在备选区生成 3 个积木。
- [x] 必须将备选区 3 个积木全部放入场地后，才会生成下一批新的积木。

## 2. 积木与放置
- [x] 积木尺寸均符合设定（不超过 3x3 范围）。
- [x] 玩家可以自由选择 3 个备选积木的放置顺序。
- [x] 积木无法放置在已被占据的格子上。
- [x] 积木的任何部分都无法超出 9x9 的边界。

## 3. 消除与计分
- [x] 当任意一行被填满时，该行方块被正确清空。
- [x] 当任意一列被填满时，该列方块被正确清空。
- [x] 支持同时消除多行、多列、以及行列交叉消除。
- [x] 放置积木成功，分数正确增加 n 分（n 为该积木占用的格子数）。
- [x] 触发消除时，分数正确增加：(消除行数 + 消除列数) × 10 分。

## 4. 游戏结束控制
- [x] 当备选区剩下的任意积木都无法放入场地的任何位置时，准确触发 Game Over。
- [x] 游戏结束后弹出提示，并提供“重新开始”按钮。
- [x] 点击“重新开始”后，场地清空、分数归零、备选区重新生成 3 个新积木。
