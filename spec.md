# 9x9 积木消除游戏 (ThousandTen) 规格说明书

## 1. 游戏概述
一款基于 9x9 网格的放置消除类休闲游戏。玩家通过拖拽或点击，将系统随机生成的积木放入网格中。填满整行或整列即可触发消除。当备选区积木无法放入网格时，游戏结束。

## 2. 核心规则
- **场地尺寸**：9x9 方格。
- **积木生成**：每次在备选区随机生成 3 个积木。必须将这 3 个积木全部放置到场地后，才会刷新下一批 3 个积木。
- **放置规则**：玩家可以自由选择备选区积木的放置顺序。积木放置时不能与其他已放置的方块重叠，且不能超出场地边界。
- **消除规则**：当场地内任意一行或一列被积木占满时，该行/列将被清空。允许多行、多列同时触发消除。
- **结束条件**：当备选区剩余的任意一个积木都无法放入场地中的任何合法位置时，判定为 Game Over。

## 3. 计分机制
- **基础放置分**：成功放置一个占据 n 个格子的积木，积分增加 n 分。
- **消除奖励分**：每触发一行或一列消除，积分增加 10 分。
  - *示例*：同时消除 1 行 1 列，奖励 20 分；同时消除 2 行 1 列，奖励 30 分。
- **总分计算**：放置分 + 消除分。

## 4. 积木样式设计 (已决策)
所有积木被限制在 3x3 的最大尺寸内。采用 **方案A（经典平衡版）**：
- 包含 1x1点、1x2/1x3直线、2x2/3x3正方形、2x2 L型、3x3 L型。难度适中，最经典的玩法。

## 5. 技术栈建议 (已决策)
- 前端框架：React 18 + Vite
- 样式方案：Tailwind CSS
- 状态管理：React Context 或 Zustand
- 拖拽交互：原生 HTML5 Drag and Drop API 或 dnd-kit
