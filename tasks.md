# 里程碑 M3｜任务清单 (Tasks)

## 阶段一：范围冻结与基础对齐
- [x] 1.1 固化 M3 范围约束：单机、Endless、稳定可控，不引入关卡/社交/商业化。
- [x] 1.2 校验并锁定当前计分规则，确保后续“连击反馈增强”不改分值模型。
- [x] 1.3 梳理现有拖拽、消除、补货流程中的反馈触发点，输出统一事件流。

## 阶段二：连击反馈系统（不加分）
- [x] 2.1 设计连击状态机：连击计数、断连判定、重置时机。
- [x] 2.2 实现连击视觉分层反馈（基础连击、强化连击、高能连击）。
- [x] 2.3 增加连击文案与短暂提示动效，避免遮挡核心棋盘信息。
- [x] 2.4 实现“连击不加分”保护逻辑与回归测试，防止误改计分。

## 阶段三：音效系统
- [x] 3.1 增加落子音、消除音、连击音阶资源与触发逻辑。
- [x] 3.2 实现音效混音策略，避免短时间高频触发导致爆音或叠音失真。
- [x] 3.3 增加音效开关与音量档位配置，并持久化本地设置。
- [x] 3.4 完成静音场景与快速连击场景的专项验证。

## 阶段四：触觉反馈系统（移动端）
- [x] 4.1 封装振动能力检测与降级处理，确保不支持设备平稳运行。
- [x] 4.2 实现落子、消除、连击的差异化振动节奏与强度映射。
- [x] 4.3 增加振动开关并与音效设置保持一致的配置体验。
- [x] 4.4 完成 iOS/Android 主流浏览器兼容性验证与兜底。

## 阶段五：移动端拖拽可视性优化
- [x] 5.1 为触控拖拽引入“积木上移偏移”能力，减少手指遮挡。
- [x] 5.2 将上移量改为相对单格尺寸的比例参数，支持不同屏幕密度。
- [x] 5.3 限定偏移仅触控生效，鼠标拖拽保持原有对齐逻辑。
- [x] 5.4 校验偏移后落点预览、合法性判定与释放放置的准确性。

## 阶段六：体验一致性与性能
- [x] 6.1 对齐视觉、音效、振动触发时序，形成统一爽感节奏。
- [x] 6.2 压测连续消除场景，确保拖拽与动画不卡顿、不掉帧。
- [x] 6.3 复测 Game Over 判定、重开流程、长局稳定性。
- [x] 6.4 完成桌面端与移动端的完整回归，并记录已知限制。
